## 一、流程控制

### 1. if-else

* if 后面的条件可以省略小括号
* 条件后面的大括号不可以省略

```swift
let age = 4
if age >= 22 {
    print("大于 22 岁")
} else if age >= 18 {
    print("大于 18 岁")
} else if age > 7 {
    print("大于 7 岁")
} else {
    print("孩子")
}
```

* if 后面的条件只能是 Bool 类型

```swift
if age == 4 {
    print("4 岁")
}
```
